# Vue 知识点

# Vue 实例

  • 每个 Vue 应用都是通过用 Vue 函数创建一个新的 Vue 实例开始的
  • 所有的 Vue 组件都是 Vue 实例
  • 只有当实例被创建时就已经存在于 data 中的属性才是响应式的
  • 使用 Object.freeze() 会阻止修改现有的属性，不再有响应式特性
  • 每个 Vue 实例在被创建时都要经过一系列的初始化过程
